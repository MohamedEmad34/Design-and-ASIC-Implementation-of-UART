* One of the biggest challenges in SOC design is the on-chip communication between the different components.
* The different bus protocols used for interconnection.
* Most of the time, the IP cores are designed with many different interfaces and communication protocols, and this can be a problem while integrating into an SOC.
* To avoid this problem, standard on-chip bus structures and protocols were developed. Exchange of information from one system to another system with a medium is       called communication.
* Universal asynchronous receiver-transmitter (UART) is one of the simplest and oldest forms of device-to-device digital communication. You can find UART devices as    a part of integrated circuits (ICs) or as individual components.
* It transfers and receives data serially, bit by bit, without clock pulses.
* The UART takes bytes of data and sequentially sends the individual bits.
* It uses a single data line for transmitting and receiving the data. It has one start bit, 8-bit data, and one stop bit, meaning the 8-bit data transfer signal     is high to low.
